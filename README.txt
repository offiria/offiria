2- What is Joomla?
    * Offiria it's a Social Network System (CMS) which enables you to build your social network and microblog.
    * It's a free and OpenSource software, distributed under the GNU General Public License version 2 or later
    * This is a simple and powerful social network cms and it requires a server with PHP and MySQL to run it. 
	More details here: http://www.joomla.org/about-joomla.html
